# forge

create go projects

## Usage

```bash
./forge --help
NAME:
   forge - A simple Go project initializer

USAGE:
   forge [global options] command [command options]

VERSION:
   0.1.0

COMMANDS:
   new      Creates a new project
   help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --help, -h     show help
   --version, -v  print the version
```
